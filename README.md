# After School – Frontend (Vue.js)

Static Vue 2 site hosted on **GitHub Pages**.
Talks to the Express/MongoDB Atlas backend using `fetch` with promises.

## Pages

- `index.html` — lesson list, sorting, Add to Cart, cart toggle button
- `cart.html`  — cart items + checkout form (name + phone validation)

## Run locally

Just open `index.html` in a browser (or serve with any static server).
Backend is expected on `http://localhost:3000`; on production the code points at the Render URL.
